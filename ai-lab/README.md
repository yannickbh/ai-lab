# CrewAI Lab

A test project for experimenting with CrewAI - a framework for orchestrating role-playing, autonomous AI agents.

## Setup

1. Install dependencies:
```bash
pip install crewai python-dotenv
```

2. Create a `.env` file (if needed) with your API keys:
```
OPENAI_API_KEY=your_key_here
```

3. Run the test:
```bash
python test.py
```

## Project Structure

- `test.py` - Simple CrewAI agent test
- `.gitignore` - Git ignore rules for Python projects
